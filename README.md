# finance-daisy

本项目是后台管理框架，基于 vue3 + vite + tailwind css + daisy ui 开发，体积小速度快。

#### 注意！！ daisy ui 组件库是一个基于tailwind css 上纯css组件库，不包含任何js内容。所有js部分需要自己动手哦！
